Thirsty Student Shop (The Steak Shop)
This project is a Node.js web application built using Express and EJS templates.
It was created for a Web Technologies lab and simulates a small online drinks shop.

What the project does
The website allows users to:
View information about the shop
Search for drinks
Register with their name
Complete a customer survey
See lists of shop locations and managers

Technologies used
Node.js
Express.js
EJS (Embedded JavaScript templates)
HTML and CSS
Forever (to keep the app running)
Git and GitHub (for version control)

How to run the app
Open a terminal in the project folder.
Run the command:
npm install express ejs
Start the server:
node index.js
or use Forever to keep it running:
forever start index.js
